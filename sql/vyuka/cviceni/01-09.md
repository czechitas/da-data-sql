Vypiš počet teroristických útoků, které se staly v roce 2015 (15138).

---solution

a)

  ```sql
       SELECT COUNT(*)
       FROM TEROR
       WHERE IYEAR = 2015;
       -- 15138
  ```

b)

  ```sql
       SELECT COUNT(*)
       FROM TEROR
       WHERE DATE_PART(YEAR, IDATE) = 2015;
       -- 15138
  ```
