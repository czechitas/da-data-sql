Vypiš počet teroristických útoků, které se staly po roce 2015

---solution

a)

    ```sql
       SELECT COUNT(*)
       FROM TEROR
       WHERE IYEAR=2015;
    ```

b)

    ```sql
       SELECT COUNT(*)
       FROM TEROR
       WHERE DATE_PART(YEAR, IDATE) = 2015;
    ```
