Kolik událostí se stalo ve třetím a čtvrtém měsíci a počet mrtvých teroristů není NULL?

---solution

```sql
SELECT COUNT(*)
FROM TEROR
WHERE IMONTH BETWEEN 3 AND 4
    AND NKILLTER IS NOT NULL;
```

```sql
SELECT COUNT(*)
FROM TEROR
WHERE IMONTH IN (3, 4)
    AND NKILLTER IS NOT NULL;
```

```sql

SELECT COUNT(*)
FROM TEROR
WHERE (
        IMONTH = 3
        OR IMONTH = 4
    )
    AND NKILLTER IS NOT NULL;
```
