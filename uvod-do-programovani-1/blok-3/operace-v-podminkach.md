## Operace v podmínkách

### Operátory porovnání

- `>`
- `>=`
- `<`
- `<=`
- `===`
- `!==`

### Příklad použití

```js
let a = 7
let b = 12
if (a === b) {
	console.log('Podmínka platí. a se rovná b.')
}
```

Výpis do konzole se neprovede, protože `7` se nerovná `12`. Neplatí podmínka, že `a === b`.
